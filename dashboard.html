Project Purpose

Our project addresses the critical challenge of optimizing resource allocation in 5G networks to improve latency performance while maintaining efficient resource utilization.

The Challenge

5G networks promise ultra-low latency, high bandwidth, and support for massive connectivity. However, these requirements often conflict:

  • Resource Competition: Multiple services compete for limited network resources
  • Diverse Service Requirements: Different applications have varying latency, bandwidth, and reliability needs
  • Dynamic Network Conditions: User mobility and varying traffic patterns create constantly changing demands
  • Efficiency vs. Performance: Static resource allocation is inefficient, while dynamic allocation risks SLA violations
The 5G Resource Allocation Challenge

Network slicing creates virtual networks tailored to specific use cases, but optimal resource allocation remains challenging.
